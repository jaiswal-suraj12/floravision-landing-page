# 🌿 FloraVision - Landing Page

A high-fidelity, responsive plant shop landing page built as part of the SoftDef Frontend Internship Screening Task.

---

## 🔗 Live Demo
[Explore FloraVision Live](https://floraplantvision.netlify.app)

---

## ✨ Features

- Modern UI/UX with clean glassmorphism design
- Fully responsive for Desktop, Tablet, and Mobile
- Smooth animations and interactive sections
- Trending Plants section
- Top Selling Products
- Customer Reviews section

---

## 🛠 Tech Stack

- React.js (Vite)
- Tailwind CSS
- Lucide React
- Framer Motion

---

## 📦 Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/jaiswal-suraj12/floravision-landing-page.git
```

### 2. Navigate into the project folder

```bash
cd floravision-landing-page
```

### 3. Install dependencies

```bash
npm install
```

### 4. Run the development server

```bash
npm run dev
```
